# Data Preparation

1. Create a folder named `SynSeg_dataset`.

2. Download the dataset from [Jarch-ma/FSS_Dataset](https://huggingface.co/datasets/Jarch-ma/FSS_Dataset).

3. Unzip the downloaded dataset into the `SynSeg_dataset` folder.
